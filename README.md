# crop_yeild_app
🌾 Crop Yield Prediction System

An end-to-end Machine Learning web application that predicts crop yield using environmental and soil parameters, enhanced with real-time weather data integration.

---

🚀 Live Demo

👉 https://cropyeildapp-iihkteodesu2uoonrnxpld.streamlit.app/

---

📌 Project Overview

This project aims to assist farmers and agricultural planners by predicting crop yield based on key parameters such as rainfall, temperature, humidity, and soil nutrients (N, P, K).

It combines machine learning with real-time weather data and intelligent recommendations to support better decision-making in agriculture.

---

⚙️ Features

- 🌾 Crop Yield Prediction using Machine Learning
- 🌦 Real-time Weather Data (Temperature, Humidity, Rainfall)
- 🌱 Fertilizer Recommendation based on NPK values
- 📊 Yield Classification (Low / Medium / High)
- 📈 Visualization of Yield Trends
- 🌐 Multi-language Support
- 📱 Responsive Dashboard UI

---

🧠 Technologies Used

- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
- OpenWeatherMap API
- Matplotlib

---

📊 How It Works

1. Select State and Crop
2. System auto-fills city for weather data
3. Fetch real-time weather parameters
4. Input soil nutrients (Nitrogen, Phosphorus, Potassium)
5. Predict crop yield using ML model
6. View:
   - Predicted yield
   - Yield category
   - Fertilizer recommendations
   - Yield trend graph
